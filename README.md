# ArtAi - San'at Yaratuvchi Platforma

Bu zamonaviy veb-ilova foydalanuvchilarga rasm yoki fayllarni yuklab, ularni san'atli chizmalar va eskizlarga aylantirish imkonini beradi.

## Xususiyatlar

- **Drag & Drop yuklash**: Rasm fayllarini tortib qo'yish orqali yuklang
- **4-bosqichli transformatsiya**: Asl rasm → Yengil eskiz → Batafsil chizma → Yakuniy san'at asari
- **Zamonaviy dizayn**: Glassmorphism va neumorphism uslublarida
- **Silliqlik animatsiyalar**: Bosqichlar o'rtasida silliq o'tishlar
- **Futuristik interfeys**: Toza, minimal va professional ko'rinish

## Ishga tushirish

1. `index.html` faylini brauzerda oching
2. Rasm faylini yuklash maydoniga tortib qo'ying yoki "tanlang" tugmasini bosing
3. "San'at yaratish" tugmasini bosing
4. Transformatsiya jarayonini tomosha qiling

## Fayllar

- `index.html`: Asosiy HTML struktura
- `style.css`: CSS stillari va animatsiyalar
- `script.js`: JavaScript funksionalligi

## Texnologiyalar

- HTML5
- CSS3 (Glassmorphism, Flexbox, Animations)
- Vanilla JavaScript

## Muammolar va yechimlar

- **Fayl yuklanmaydi**: Faqat rasm fayllarini (.jpg, .png, .gif) qo'llab-quvvatlaydi
- **Animatsiya ishlamaydi**: Zamonaviy brauzerda oching (Chrome, Firefox, Safari)
- **Mobil ko'rinish**: Responsive dizayn mavjud, lekin eng yaxshi desktopda ko'rinadi

Bu demo versiya bo'lib, haqiqiy AI transformatsiyasi uchun backend integratsiyasi kerak.
